# ws i (インタラクティブモード)

fzf を使って対話的にコマンドを組み立てて実行します。

## 使い方

```bash
ws i
```

## 動作

1. fzf でコマンド一覧を表示
2. コマンドを選択すると、必要なパラメータを対話的に入力
3. 組み立てたコマンドを `ws` のサブプロセスとして実行

## 選択可能なコマンド

| コマンド | 説明 |
|---------|------|
| `clone` | bare リポジトリを作成 |
| `new` | workspace を作成 |
| `rm` | workspace を削除 |
| `list` | worktree 一覧表示 |
| `status` | 全体の状態表示 |
| `shared` | 共有ファイル管理（サブメニューあり） |

## 依存関係

インタラクティブモードには [fzf](https://github.com/junegunn/fzf) が必要です。`fzf` がインストールされていない場合はエラーになります。

## 例

```bash
$ ws i
# → fzf でコマンド選択画面が表示される
# → "new" を選択
# → 名前、場所、ブランチ、起点を対話的に入力
# → 最終的に "ws new feature/foo --from main" が実行される
```

実行前に組み立てたコマンドが stderr に表示されるため、何が実行されるか確認できます。

```
> ws new feature/foo --from main
```
